# Console Password Manager

**Автор:** [Игорь Прокопьев]

## Описание
Console Password Manager — это легковесное консольное приложение для локального хранения и генерации паролей. Программа построена с использованием паттерна MVC и принципов ООП. Данные надежно сохраняются в формате JSON на локальном диске.

## Функционал
- Добавление, просмотр и удаление записей паролей.
- Генерация безопасных случайных паролей (настраиваемая длина и спецсимволы).
- Автоматическое сохранение данных в `passwords.json`.
- Защита от некорректного ввода (валидация пустых полей).
